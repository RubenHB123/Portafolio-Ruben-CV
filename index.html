Proyectos

MotorCity (Concesionario de motos)

Proyecto final que hice en el grado superior de ASIR. Trata sobre un concesionario de motos, con formulario de registro, reservas, login y logout, filtro para la busqueda de motos. Creado con HTML, CSS, SQL, PHP, PhpMyAdmin y XAMPP.


Web de un Festival de Música

Proyecto sobre una pagina web de un Festival de Música. El proyecto es una única página con navegación, distintas animaciones y uso de JS y CSS para diferentes apartados. Creado con HTML, CSS, JavaScript, SASS, NPM, JSON y GULP.
